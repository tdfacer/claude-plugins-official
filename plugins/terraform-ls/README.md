# terraform-ls

Official HashiCorp Terraform language server for Claude Code, providing code intelligence, validation, and formatting for Terraform and HCL files.

## Supported Extensions
`.tf`, `.tfvars`, `.tfstack.hcl`, `.tfcomponent.hcl`, `.tfdeploy.hcl`, `.tfquery.hcl`

## Installation

Install terraform-ls using one of the following methods:

### macOS / Linux (Homebrew)
```bash
brew install hashicorp/tap/terraform-ls
```

### Arch Linux
```bash
sudo pacman -S terraform-ls
```

### Ubuntu/Debian
Follow the [HashiCorp official packaging guide](https://www.hashicorp.com/official-packaging-guide) and install the `terraform-ls` package.

### From releases
Download pre-built binaries from [HashiCorp releases](https://releases.hashicorp.com/terraform-ls/).

Make sure the `terraform-ls` executable is in your PATH.
